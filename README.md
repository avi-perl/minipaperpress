# MiniPaperPress

MiniPaperPress is a browser-based tool for laying out documents that fold down into something pocket-sized such as a
credit-card-sized packet that unfolds into a long easy-to-carry notepad.

Check it out: {url}

## Features

- **Print** straight from the browser, with multiple copies tiled onto a standard letter sheet and cut/fold guides
  included.
- **Export to PDF** for archiving or sending to a print shop.
- **Share** your designs with a link so others can view, print, or remix them.
